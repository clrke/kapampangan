# 🇵🇭 Kapampangan Cheat Sheet

A beginner-friendly cheat sheet for **reading and replying in Kapampangan** (the language of Pampanga, Philippines), built as a single static page for GitHub Pages.

👉 **Live site:** https://clrke.github.io/kapampangan/

## What's inside

- **Core vocabulary** — the short connective words that trip up beginners (`ngeni`, `eku`, `metong`, `abe`…).
- **Grammar patterns** — pronoun clusters, negation with `E`, and how borrowed English/Tagalog words fit in.
- **Handy reply phrases** — greetings, "get well soon," "take care," "see you," and more.

## Develop locally

It's a single `index.html` — just open it in a browser:

```sh
open index.html
```

## Privacy

Private chat transcripts are **never committed**. The following are git-ignored (see `.gitignore`):

- `private/`, `conversations/`
- `*.private.md`, `*.chat.txt`

Keep any personal conversation logs inside those paths so they stay off GitHub.
